## 🧭 FootballPad 전체 구조 (통합 개요)

```
FOOTBALLPAD
 ┣━ ⚙️ 리그 관리 (League Management)
 ┣━ 📅 경기 관리 (Match Management)
 ┣━ 👥 팀 관리 (Team Management)
 ┣━ 🧑‍🤝‍🧑 선수 관리 (Player Management)
 ┣━ 🎬 영상/통계 분석 (Bepro-style)
 ┣━ 📊 데이터 리포트 (WhoScored-style)
 ┣━ 🧠 전술보드 (TacticalPad-style)
 ┣━ 💬 커뮤니티 / SNS
 ┗━ 🏆 리그/시즌 통계 & 랭킹
```

## ⚙️ 1. 리그 관리 (League Management)

**🎯 기능 목적:**
리그를 생성하고 시즌, 경기 주기, 경기 수, 참가 팀 등을 체계적으로 관리.

| 기능       | 설명                                            |
| -------- | --------------------------------------------- |
| 리그 생성    | 리그 이름, 시즌 기간(시작~종료일), 경기 주기(예: 주 1회), 경기 수 입력 |
| 리그 설정    | 경기 규칙(승점, 득실차, 무승부 점수 등), 경기 장소, 경기 시간대       |
| 참가팀 관리   | 팀 등록/초대, 승인/거절                                |
| 시즌 캘린더   | 주차별 경기 자동 배정 or 수동 스케줄링                       |
| 리그 홈화면   | 순위표, 일정표, MVP, 뉴스 요약 표시                       |
| 시즌 종료 기능 | 자동 통계 생성 (득점왕, 어시스트왕, MVP 등)                  |

**데이터 구조 예시**

```json
{
  "league_id": "HPL2025",
  "name": "Hosanna Premier League 2025",
  "season_start": "2025-09-01",
  "season_end": "2026-01-31",
  "game_cycle": "weekly",
  "max_games": 20,
  "teams": ["믿음", "소망", "사랑"],
  "rules": { "win": 3, "draw": 1, "loss": 0 },
  "status": "active"
}
```

| ----- | ----------------- |

| 시즌 기간 | 날짜 선택 (start–end) |

### 📊 [리그 대시보드 화면]

| 리그 헤더    | 이름, 시즌, 상태           |
| 진행상황 위젯  | 경기수, 잔여경기            |
| 순위표 미리보기 | TOP 3 팀              |
| 일정 미리보기  | 이번 주 경기              |

## 📅 2. 경기 관리 (Match Management)

### 📋 [경기 일정표 화면]

**경로:** `/leagues/:id/matches`

* 캘린더 또는 리스트 뷰
* 필터: 라운드별 / 날짜별 / 팀별
* 경기 카드:

  * 홈 vs 어웨이
  * 날짜, 장소
  * 결과(입력 전이면 “예정” 표시)
  * 영상 업로드 상태 아이콘 🎥

---

### 🧾 [경기 상세 화면]

**경로:** `/matches/:id`

| 구성    | 설명                    |
| ----- | --------------------- |
| 상단 요약 | 팀 로고 / 스코어 / 날짜 / 경기장 |
| 영상 탭  | 전체 영상 / 하이라이트 / 클립 공유 |
| 기록 탭  | 득점자, 어시스트, 카드         |
| 통계 탭  | 점유율, 슈팅, 패스 등 그래프     |
| 댓글 탭  | 피드백, MOM 투표, 좋아요      |

---

### 🎥 [영상 업로드 화면]

**경로:** `/matches/:id/video-upload`

* 파일 드래그 & 드롭
* 자동 썸네일 생성
* 클립 구간 지정 (시작–끝)
* 태그 추가 (득점, 슈팅, 세이브 등)
